# 1897. Redistribute Characters to Make All Strings Equal
###### tags: `LeetCode`
## **Link**
https://leetcode.com/problems/redistribute-characters-to-make-all-strings-equal/
## **Code**
```cpp=
class Solution {
public:
    bool makeEqual(vector<string>& words) 
    {
        vector<int> ct(26,0);
        int n=words.size();
        for(string v:words)
            for(char w:v)
                ++ct[w-'a'];
        for(int v:ct)
            if(v%n)
                return false;
        return true;
    }
};
```
