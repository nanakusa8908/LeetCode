# 2544. Alternating Digit Sum
###### tags: `LeetCode`
## **Link**
https://leetcode.com/problems/alternating-digit-sum/
## **Code**
```cpp=
class Solution {
public:
    int alternateDigitSum(int n) 
    {
        string s=to_string(n);
        int ans=0;
        for(int i=0;i<s.size();++i)
            if(i&1)
                ans+=(-1)*(s[i]-'0');
            else
                ans+=s[i]-'0';
        return ans;
    }
};
```
